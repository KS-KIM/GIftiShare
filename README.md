# GiftiShare

이더리움 스마트 컨트랙트를 기반으로 쿠폰, 기프티콘 등의 자산을 공유할 수 있는 안드로이드 어플리케이션입니다.

## Features
- 이더리움 지갑 생성
- 이더 조회 및 전송
- 트랜잭션 조회
- 쿠폰 등록
- 쿠폰 구매
- 구폰 구매 및 판매목록 조회
